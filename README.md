# oxo-wallet
* use gen.js to generate message  
0、Declare  
1、Payment  
2、SubjectProclaim  
3、SubjectRevoke  
4、TrustCreate  
5、TrustRemove  
6、OfferCreate  
7、OfferCancel  

* use jp.html to send Transaction to Server  
click 【Connect】 to connect  
first 【Send】 Declare（0） message  
then 【Send】 Transaction（1/2/3/4/5/6/7） message to Server
